OffTheWorldlyRoad Architecture

Next.js 16 + Auth.js + Vercel Postgres

Frontend Layer

  • • Next.js 16 App Router
  • • React 19 + Tailwind CSS v4
  • • Server Components by default
  • • Client components for interactivity

Auth & Identity

  • • Auth.js v5 (NextAuth successor)
  • • OAuth 2.0 via GitHub / Google
  • • JWT sessions stored in HTTP-only cookies
  • • Account profiles in Postgres

Data Layer

  • • Vercel Postgres (Neon serverless)
  • • Prisma ORM for type-safe queries
  • • Users, Accounts, Sessions, Profiles
  • • Auto-scaling connection pool
